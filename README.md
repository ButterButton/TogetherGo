# TogetherGo 團購機器人
一、說明
1. 簡介
	* 許多人喜歡使用Line的群組揪團購，但一筆一筆複製字串沒效率且可能發生衝突，所以發想出用聊天機器人來幫忙處理訂單記錄的工作。
2. 開發語言
	* Python
3. 架構
	* Flask
4. 套件
	* Line - Messaging API
	* Line - LIFF
	* Virtualenv
5. 資料庫
	* MySQL
